# 🌦️ WeatherNow (Work in Progress 🚧)

This is a weather web application currently under development. The goal of this project is to build a responsive and real-time weather app using modern web technologies.

---

## 🚧 Project Status

⚠️ This project is currently under development. New features are being added regularly.

---

## 🎯 Features (Planned / In Progress)

* 🌍 Search weather by city name
* 🌡️ Display temperature and weather conditions
* 💨 Show humidity and wind speed
* 📱 Responsive UI
* 📍 Current location weather (coming soon)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* React.js
* Weather API
